# `tack` Project Manager

`tack` is a TUI project manager that utilizes `notcurses` as its frontend,
`PostgreSQL` as its backend, and is multi-threaded.

# Build

## Requirements

1. **PostgreSQL v18.3-1+**
2. **libpg_query v17.6.2.2-1+** (Arch)
3. **GCC**
4. **GNU Make**

## Building

```bash
$ make release
$ make install
$ tack
```
